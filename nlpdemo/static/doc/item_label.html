商品标签

       商品标签服务 提供商品描述的打标签功能，对输入的商品标题，返回该商品的各维度标签。

URL

       http://nlp.api.baifendian.com/item_label

HTTP Method

       支持GET、POST

HTTP query string参数

       本服务需要传递的参数包括：

  • title：商品描述性标题；
  • token：用户token

HTTP 请求 Body

        输入为string类型的商品标题和token。比如：

        {"title":"24寸个性时尚双色拉杆箱前黄后蓝","token":"YOUR_API_TOKEN"}

HTTP 返回 Body

       字典类型的返回结果，key值包括：status、modTime、allTime、result、uuid。其含义分别为：

  • status：状态码
  • result：分析结果。分析结果是一个json类型的数据，其中keys包括：brand_name_new（品牌，若其value为空，则不返回此key）、category_name_new（类别，若其value为空，则不返回此key）、attr（维度标签，若其value为空，则不返回此key）
  • modTime：rest服务通过rpc调用服务的时间，单位是ms。
  • allTime：rest服务处理时间，单位是ms。
  • uuid：此次请求的标识，用来定位会话以及问题。

Python 调用示例

url = "http://nlp.api.baifendian.com/item_label"
title = """
24寸个性时尚双色拉杆箱前黄后蓝
"""
data = {"title":title,"token":"YOUR_API_TOKEN"}
resp = requests.post(url,data=data)
