自动摘要

       自动摘要服务 提供中文文本的自动摘要提取功能，对输入的文本内容，返回文本的摘要信息。

URL

       http://nlp.api.baifendian.com/auto_summary

HTTP Method

       支持GET、POST

HTTP query string参数

       本服务需要传递的参数包括：

  • text：待分析的文本；

  • title：文本的标题，可以省略；

  • language：语言，支持chinese和english；默认为chinese；

  • token：用户token。

HTTP 请求 Body

       输入为string类型的文本内容、string类型的标题、string类型的语言类型和token。比如：

       {"text":"大型第三方支付的数据已经具有了不同于银行的特征。", "title":"","language":"chinese", "token":"YOUR_API_TOKEN"}

HTTP 返回 Body

       字典类型的返回结果，key值包括：status、modTime、allTime、result、uuid。其含义分别为：

  • status：状态码
  • result：分析结果，即摘要结果。
  • modTime：rest服务通过rpc调用服务的时间，单位是ms。
  • allTime：rest服务处理时间，单位是ms。
  • uuid：此次请求的标识，用来定位会话以及问题。
