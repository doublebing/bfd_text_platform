使用入门

       百分点企业级文本分析平台服务以REST API的方式提供服务，支持两种调用方式，一种是通过request请求调用，另一种是通过Python SDK的方式调用。

       在正式开始前，您需要首先注册账号。完成后，您将在用户管理页面看到您的API Token （密钥），该密钥将用于身份验证。

环境依赖

       因部分API已提供公网服务，暂时内网调用需要配hosts文件，需配置的信息为：

       192.168.61.209 nlp.api.baifendian.com

       文本平台服务依赖于Python环境和requests包，其版本分别为python的2.7版本，及其对应的requests安装包。可以从以下网站中下载相应组件：

       Python安装： https://www.python.org/doc/

       requests安装：http://www.python-requests.org/en/latest/

request请求方式调用

       请参考各服务中的“Python调用示例”版块进行使用。

Python SDK调用

       Python SDK的安装包请到git中下载，见http://git.baifendian.com/congwei.liang/bfdnlp。其具体调用方式请参考各服务中“Python SDK调用示例”版块进行使用。
