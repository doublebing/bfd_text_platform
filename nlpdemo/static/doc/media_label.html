媒体标签

       媒体标签服务 提供中文文本的媒体标签功能，对输入的文本内容，返回文本的关键词和品类。

URL

       http://nlp.api.baifendian.com/media_label

HTTP Method

       支持GET、POST

HTTP query string参数

       本服务需要传递的参数包括：

  • content：待分析的文本；

  • numWords：需要提取关键词的个数；

  • title：文本的标题，可以省略；

  • token：用户token。

HTTP 请求 Body

       输入为string类型的文本内容、int类型的关键词个数、string类型的标题和token。比如：

       {"content":"自从加盟火箭队之后迈克丹尼尔斯并没有获得太多的表现机会", "numWords":2,"title":"", "token":"YOUR_API_TOKEN"}

HTTP 返回 Body

       字典类型的返回结果，key值包括：status、modTime、allTime、result、uuid。其含义分别为：

  • status：状态码
  • result：分析结果。其中包括关键词和对应的权重值、以及品类。。
  • modTime：rest服务通过rpc调用服务的时间，单位是ms。
  • allTime：rest服务处理时间，单位是ms。
  • uuid：此次请求的标识，用来定位会话以及问题。

Python 调用示例

import requests
url = "http://nlp.api.baifendian.com/media_label"
data = {"content":"自从加盟火箭队之后迈克丹尼尔斯并没有获得太多的表现机会", "numWords":2,"title":"", "token":"YOUR_API_TOKEN"}
resp = requests.post(url,data=data)
print resp.text
